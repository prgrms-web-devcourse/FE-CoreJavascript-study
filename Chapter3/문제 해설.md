## 📌문제5

### 다음 코드의 실행 결과를 작성하세요.

(this는 가리키는 객체만 작성해주세요)

```js
let obj = {
  outer: function () {
    console.log(this); // (1)
    var innerFunc1 = () => {
      console.log(this); // (2)
    };
    innerFunc1();

    let self = this;
    let innerFunc2 = function () {
      console.log(self); // (3)
    };
    innerFunc2();
  },
};
obj.outer();
```

### 답안 작성

```
(1) : outer
(2) : outer
(3) : outer
```
